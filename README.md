# Documents For MLPerf BERT Pre-Training 2023 (3.0)


## Implementations

We have implemented the following versions of BERT pretraining:
- NVIDIA's implementation
    - A5000 (2 GPUs) - owned by KRAI
    - H100 (1 GPU) - owned by lamadalabs

- HuggingFace's implementation
    - T4 (1 GPU) - owned by colab (free)
    - A5000 (2 GPUs) - owned by KRAI

# MkDoc Usage

## Install requirements
```
python3 -m pip install -r requirements.txt
```


## Run
```
python3 -m mkdocs serve
```
